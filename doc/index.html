    Game : Eyes of the World

    Author: Bartosz Marchewka

    About game

    The project is simple text RPG game. It is loosely based on a board game „Talisman”. The adventure takes place not in the fantastic land, and the game is designed for two Players. The goal of the game is to kill the other player (or survie longer). In the game it is possible to move, collect items, fight with beings and other player, character class selection, leveling, visitingthe cities with medics, alchemists, witches and shops. Like board games, the map is a closed circle. It consist of fields of 2 types:

    • “Inhabited Areas” – where players are able to for example do the shopping or heal wounds

    • “Lands” – there players will encounter monsters, but also items, useful in further journey

    The player’s inventory consists of 4 items. These are divided into weapons and armors.

    Usage

    Make sure that all files visible in Project are in the same folder as .rb files. Launch the game using:

    ruby Game.rb
